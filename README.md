1. Clone the repository:
   ```bash
   git clone https://github.com/C0dewithLokesh/LinkedInReplyWriter_AI.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Lunacal_Assignment
   ```
3. Run the following command to start the application:
   ```bash
   pnpm i
   # or
   npm i

   pnpm dev
   # or
   npm run dev
   ```
